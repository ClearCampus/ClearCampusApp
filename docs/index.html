Sign up in 30 seconds. Find a club in 2 minutes. RSVP in 1 click. Nothing existing is this fast.

A system that understands "not too serious" beats scrolling a list. Students won't go back.

Network effects

More students using it = more data on what works. More clubs updating = better descriptions. Both sides improve together.

Existing services haven't improved in a decade. This is a generational upgrade using technology that just became cheap enough to run.
